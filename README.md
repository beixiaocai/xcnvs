### 项目介绍
* 管理平台（简称xcnvs）既是一款开源支持多人协作的，样本导入+样本标注+模型训练+模型管理+模型测试+模型导出的工具。也是一款视频文搜功能，万物搜功能的AI视频分析工具，也可以用来作为视频行为分析系统v3/v4的集群管理软件。
* 相关项目
* 视频行为分析系统v4： https://gitee.com/Vanishi/xcms
* 视频行为分析系统v3： https://gitee.com/Vanishi/BXC_VideoAnalyzer_v3
* xclabel： https://gitee.com/Vanishi/xclabel
